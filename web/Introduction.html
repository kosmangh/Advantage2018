General Overview



Advantage is a property rating application for the State Housing Corporation (SHC),Kumasi. Advantage is basically a software package which aims to computerize all aspects of record keeping activities of SHC.

It has evolved over the years and it is constantly evolving to incorporate new ideas and features which will ultimately make Advantage a better rent billing software.
