Estate

This functionality enables management of estates developed by the State Housing Company (SHC). It enables the creation, updating, retrieval and deletion of estates.

Data required to create an estate includes


Data

Description

Estate Name

This shows the name given to the estate

Estate Initials

This identifies the initials of the estate

Location

This shows the location of an estate

Plot Size

This allows users to allocate plot sizes to estates

Category

This shows which particular category an estate name belongs to i.e. First Class, Second Class or Third Class
