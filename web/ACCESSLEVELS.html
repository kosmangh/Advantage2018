ACCESS LEVELS

There is the Administrator level which is the highest and the administrator has access to all the features of the application and can choose which levels and features that should be available to all the other users of the application.

The various access levels that can be created by the Administrator include but not limited to the following;

Director

Estate Officer

Accountant

Auditor

For the purposes of this manual we assume that administrator has log on.
