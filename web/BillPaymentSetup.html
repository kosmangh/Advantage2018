Bill Payment Setup

Bill Payment displays payment records for the various estate properties and the entry of payments for the various ground and rental bills for estate properties

Data Requirement:


Data

Description

Estate

This is selected from the already existing estates

Block

This enables users to select an already existing estate block

Property No

This enables users to select a property which can be found in a block


After selecting a property based on the above requirements, a button is clicked which displays lessee property details and bill payment details.
