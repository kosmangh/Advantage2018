Setting up the application

Typically, different functions to be performed demands different data set being available before commencement.

However, before any activity can be performed, first and foremost the estates and blocks should be available. This is because, every activity performed one way or the other necessitates the requirement of an estate and block.

To get things started,

  1. You create an estate i.e. estate registration. Estate registration includes assigning an estate category to a particular estate
  2. Select an estate and then assign a block name
  3. You create an estate property by selecting an existing estate name, estate block and then assigning a number to a property
  4. Create the bill payment by first loading an estate property ledger. The necessary information about that property would be displayed then you can save details on bill payment
  5. To load a ledger for a particular estate, you go to estate ledger and select an estate, block, property no. and then a year and click on 'Load Ledger' to display results.
