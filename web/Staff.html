Staff

This functionality enables registration of staff.


In order to register a staff:

1. Enter the surname

2. Indicate other names

3. Provide phone number and alternative ones if any

4. Enter address in the space provided

5. Select a status which can be either 'Active' or 'Inactive'

6. Lastly press on 'Save' so as to save all details
